This is a landing page designed with Bootstrap. It's a great way to use only one page for both your "full site" and mobile site all in one.

It's very easy to customize the CSS so certain elements appear however you'd live, depending on the screens size. You can even make it so that some things do not show at all for mobile, but do for desktop!

This is a pop-up modal

This pop-up is all inside the same index.html file. You do not need to create a separate file for a pop-up window. Use this pop-up to play a video or show some important info that doesn't need to be displayed on your page. You can even modify the modal so it displays different content or the same content but in different sizes, depending on the way it's being viewed. You gotta love responsive design!
